 MetaFont Preview macros [for WinEdt 10]

===========================================================

 by Karl Koeller
 E-mail: karlkoeller@gmail.com

 http://www.winedt.org/

___________________________________________________________


This package contains macros that simplify the editing-viewing
process of MetaFont source files in WinEdt. It provides a menu
command that outputs a PDF file containing all fonts from the
source .mf file.

---------------------------
 USAGE
---------------------------

To preview the fonts in the MetaFont .mf file you're currently
editing, first compile the document (choose "TeX" > "MetaFont" >
"MetaFont"), then choose "TeX" > "MetaFont" > "MetaFont Preview"
(or click the corresponding toolbar button). The output files are
put in a resulting <yourfile>_preview.pdf file.
